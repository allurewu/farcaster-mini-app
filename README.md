# Farcaster Mini App

这是一个简单的Farcaster Mini App示例。

## 功能特点

- 使用React构建
- 集成Farcaster Auth Kit
- 响应式设计
- 简单的用户界面

## 安装

```bash
npm install
```

## 开发

```bash
npm start
```

## 构建

```bash
npm run build
```

## 部署

构建完成后，将`dist`目录中的文件部署到您的服务器。

## 注意事项

- 确保在部署前更新`public/index.html`中的Open Graph元数据
- 需要配置Farcaster Auth Kit的相关设置
